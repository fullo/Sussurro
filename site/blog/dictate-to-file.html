Feature

Dictate straight to a file

Note-taking mode: instead of pasting into whatever app has focus, Sussurro appends each dictation to a file you choose — like an Obsidian note.

3 min read · Feature

Sometimes you're not filling in a field — you're capturing thoughts. Dictate-to-file turns Sussurro into a hands-free notebook: every dictation is appended to a chosen .md or .txt file, no matter which window is in front.

How to use it

  1. Open the Behavior section.
  2. Turn on dictate-to-file and pick a target file — e.g. a note in your Obsidian vault.
  3. Dictate. Each result is appended to that file instead of pasted into the focused app.
Great with a vault

Point it at a Markdown note in Obsidian, Logseq, or any folder your notes app watches, and you have a frictionless capture inbox — talk, and it's in your notes, cleaned up, waiting for you.

Still the full pipeline

Everything else applies: cleanup, translation, your dictionary and snippets. The only difference is the destination — a file on disk rather than the cursor. And like the rest of Sussurro, it all happens locally.
