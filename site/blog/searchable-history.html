Feature

Searchable history & retention

Every dictation is saved locally — both the raw transcript and the cleaned result — so you can find it, reuse it, reshape it, or set it to expire.

5 min read · Feature

Dictation is ephemeral by nature — you speak, it pastes, it's gone. Sussurro keeps a local record so nothing is truly lost. History lives in a plain JSONL file on your machine; it never leaves.

What each entry holds

Both the raw transcript and the cleaned output, so you always have the verbatim version alongside the polished one.

What you can do with an entry

Hover any entry to:

  • Copy the cleaned text.
  • Re-clean the raw transcript at the current cleanup level.
  • Translate it into another language.
  • Edit it — which also feeds the self-learning dictionary.

Search & retention

Full-text search runs over the whole history — raw and cleaned. Retention auto-deletes entries older than a number of days you set (0 keeps everything forever). Export the lot to Markdown or JSON from the card header.

Usage stats that stick

Persistent counters sit atop the history: total, today, and last-7-days dictations and words. Clearing or pruning your history never resets them, so your all-time totals keep counting.

Because it's a local JSONL file, your history is also scriptable through the local API — query recent entries or search from any tool.
