Building Sussurro on Windows, macOS & Linux

Sussurro is a Tauri 2 app — a Rust backend with a React front-end. The build is the same everywhere; only the prerequisites and a few GPU details differ per platform.

7 min read · Setup guide

Most people should just grab an installer from the releases page. But if you want to build it yourself — to hack on it, or to compile the optional Linux GPU variant — here's the map. The authoritative, always-current guides live in the repo under docs/compile/; this is the overview.

Common toolchain

Every platform needs the same two things:

  • Node.js ≥ 24
  • Rust stable (via rustup)

Then the universal build steps:

git clone https://github.com/fullo/Sussurro
cd Sussurro/sussurro
npm install
npm run tauri dev      # development, hot-reload
npm run tauri build    # production bundle (installer per platform)

Windows 10/11

  • Vulkan SDK — for GPU-accelerated Whisper.
  • LLVM — libclang is needed by the Rust bindings; set LIBCLANG_PATH.
  • WebView2 — usually already present on Windows 11.
  • The MAX_PATH gotcha: whisper.cpp's Vulkan shader sub-build produces very long paths. Set a short CARGO_TARGET_DIR (e.g. C:\sbuild) or the build fails with MSBuild FTK1011.

Full details: docs/compile/windows.md.

macOS 11+

  • Apple Silicon only. ONNX Runtime has no prebuilt binaries for Intel Macs, so the Parakeet engine can't ship there.
  • Metal provides GPU acceleration out of the box.
  • Permissions: macOS will ask for microphone and accessibility access on first run (the latter for paste injection).
  • Minimum 11.0 — whisper.cpp itself only needs macOS 10.15 (for std::filesystem); the 11.0 floor is the Apple Silicon (arm64) baseline.

Full details: docs/compile/macos.md.

Linux

  • glibc ≥ 2.38 (Ubuntu 24.04 or newer) — older glibc fails linking.
  • apt dependencies — WebKitGTK and friends; the guide lists the exact packages.
  • X11 & Wayland: injection uses the XDG RemoteDesktop portal first on Wayland, with ydotool/wtype fallbacks.
  • CPU-only by default. An opt-in Vulkan build exists — compile with --features linux-vulkan if you have a Vulkan driver.

Full details: docs/compile/linux.md.

Tests & CI

From sussurro/src-tauri: cargo test and cargo clippy --all-targets -- -D warnings. See docs/development.md for the CI setup and the local-CI script.

Contributing
