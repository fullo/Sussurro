Voice & formatting commands

Structure shouldn't depend on a language model guessing. Say the word and Sussurro inserts the exact line break or bullet — deterministically.

4 min read · Feature

When you want a new line or a bullet point, you want it every time — not most of the time, subject to a model's mood. Sussurro handles formatting commands deterministically, before any LLM is involved, so structure is reliable.

Formatting commands

These are recognized in English and Italian:

  • “new line” / “a capo” / “nuova riga” — a line break.
  • Paragraph variants — a blank-line paragraph break.
  • “period new line” / “punto e a capo” — ends the sentence and breaks the line.
  • “new bullet” / “punto elenco” — starts a bulleted item.

These never reach the model — they're applied literally, so a stray “um” in a sentence can't turn your bullet into prose.

Contextual commands

Some spoken edits are best handled by the cleanup model because they depend on meaning — like “scratch that”, which deletes the phrase you just said. These ride the cleanup prompt rather than being applied literally, so they understand context.

Two kinds, on purpose

Deterministic commands (line breaks, bullets) run without the LLM for reliability. Contextual commands (“scratch that”, quote wrapping) go through cleanup, because they need to understand what you meant.

Combine these with voice snippets and you can dictate structured text — lists, multi-paragraph notes — without touching the keyboard.
