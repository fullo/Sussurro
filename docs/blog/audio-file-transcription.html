Feature

Transcribe audio files

Not everything is live. Drop in a recording and Sussurro runs it through the exact same speech-to-text and cleanup as your dictation.

3 min read · Feature

Voice memos, an interview, a meeting recording — Sussurro can transcribe files, not just your microphone. It's the same pipeline, so the output matches the quality and style you already get from live dictation.

Supported formats

wav, mp3, m4a, flac, ogg.

How to use it

  1. Open the Audio file card.
  2. Choose a recording.
  3. Sussurro transcribes it with your chosen engine and applies your cleanup settings.
Same engines, same privacy

File transcription uses Whisper or Parakeet exactly like live dictation — and stays entirely on your machine. Your recordings are never uploaded.

Prefer scripting?

The local API exposes the same capability at /transcribe, so you can batch-process a whole folder of recordings from a script.
