Setup

First run: from install to your first dictation

Install, clear the Setup banner, pick a speech model, set your shortcut, and make your first dictation — with or without Ollama running.

5 min read · Setup guide

Sussurro opens on the Settings screen the very first time you launch it — before you dictate a word, the app wants you to know exactly what's ready and what isn't. That's the job of the Setup banner: a checklist at the top of the window that lists anything missing, each item with a one-click fix.

What the Setup banner checks

The banner only appears when something needs attention, and disappears once everything is green. It can flag:

  • Accessibility permission not granted (macOS) — Sussurro needs it to paste dictated text into other apps. An Open Settings button jumps straight to the right pane.
  • Microphone access denied — dictation can't record until you allow it; same one-click jump.
  • Ollama not installed — cleanup and translation need it, but dictation still works with the raw transcript. A Get Ollama button opens the download page.
  • Ollama installed but not running — start the Ollama app (or run ollama serve), then hit Re-check.
  • The cleanup model isn't pulled yet — a Pull it button runs ollama pull for you.
  • The speech model isn't downloaded — a Download button starts it directly from the banner.

You can dismiss the banner at any point; it comes back if you reopen the app with something still unresolved.

NO OLLAMA, NO PROBLEM

Ollama only powers the AI cleanup step — filler removal, punctuation, tone, translation. If it's not installed or not running, Sussurro automatically falls back to the raw transcript instead of failing. You can also set Cleanup to "None" yourself if you'd rather skip it entirely. See cleanup levels for what each setting actually changes.

Pick a speech model

Under Dictation → Engine, the default is Whisper, GPU-accelerated on Windows (Vulkan) and macOS (Metal), and CPU-based on Linux by default (a Vulkan build is an opt-in from source). It offers a choice of model sizes — Base English at 148 MB is the fastest way to get started, Large v3 Turbo at 574 MB is the most accurate. Click Download next to your choice and wait for the progress indicator.

If you'd rather skip the size decision, switch Engine to Parakeet: a single fixed 456 MB model that's CPU-optimized (no GPU needed, roughly 10x faster than Whisper without one) and auto-detects 25 European languages. See Whisper vs. Parakeet for the full trade-off.

Set your shortcut

The default hold-to-record shortcut is Ctrl+Shift+Space (⌘⇧Space on Mac). To change it, click the hotkey field in Dictation → Shortcut — it turns into a capture widget ("Press keys… Esc to cancel") — then press the combination you want. There's a separate shortcut for command mode (default Ctrl+Alt+Space), used to edit selected text instead of dictating new text.

Push-to-talk or toggle

  1. With push-to-talk on (the default), hold the shortcut or the header Dictate button like a walkie-talkie: speak while held, release to finish.
  2. Turn push-to-talk off in Behavior if you'd rather tap once to start and again to stop.
  3. Either way, a small recording overlay appears near the bottom of the screen — red and pulsing while listening, a spinner while transcribing.

Grant permissions (macOS)

The first time you dictate, macOS asks for microphone access, and separately for Accessibility access so Sussurro can paste into whatever app has focus. If you miss either prompt, the Setup banner catches it afterward with a direct link to the right System Settings pane — you don't have to hunt for it yourself.

Your first dictation

Once a model shows as downloaded and the banner is clear (or you've decided to dictate without Ollama), click into any text field, hold your shortcut, and speak. Release, and the cleaned-up — or raw, if Cleanup is set to None — text lands wherever your cursor was, with your clipboard restored afterward. Everything above happens once. After that, closing the window tucks Sussurro into the system tray (and if you enable Launch at login in Behavior, it starts there directly), so it stays out of your way.
