Command mode: edit text with your voice

Dictation writes new text. Command mode changes text that's already there — select it, speak an instruction, and the result replaces your selection.

4 min read · Feature

Sometimes you don't want to write — you want to edit. Command mode turns any selected text into raw material for the LLM: highlight a sentence, hold the command shortcut, say what you want done, and Sussurro swaps in the rewritten version.

How it works

  1. Select some text in any app.
  2. Hold the command shortcut (default Ctrl+Alt+Space) and speak an instruction.
  3. The LLM applies it and the result replaces the selection.

Things to say

  • “Make it shorter.”
  • “Fix the grammar.”
  • “Translate to English.”
  • “Make this more formal.”
  • “Turn this into a bulleted list.”
Different from dictation

Regular dictation transforms your speech and errs on the side of pasting something. Command mode transforms the selected text using your speech as the instruction — and if it fails, it surfaces the error rather than silently pasting the untouched selection back.

Why it's handy

It's the fastest way to make a quick edit without reaching for the mouse and menus — tighten an email, fix a typo-ridden sentence, restyle a paragraph — all in the app you're already in. Command mode uses the same local model as cleanup, so nothing you select is sent anywhere.
